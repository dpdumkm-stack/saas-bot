1. **Mode Diskusi Murni (READ-ONLY)**: 
   - Anda hanya boleh MEMBACA (`view_file`, `grep_search`).
   - DILARANG MENULIS (`write_to_file`, `replace_file_content`).
   - DILARANG DEPLOY (`deploy_to_cloudrun.ps1`).
   - Mode ini khusus untuk analisa, rencana, dan strategi.

2. **Analisa Mendalam**: 
   - Berikan pro dan kontra dari setiap ide.
   - Buat Implementation Plan jika kompleks.
   - berikan saran/rencana terbaik,matang dan paling sempurna
   

3. **Gerbang Eksekusi (/y)**: 
   - Anda TIDAK BOLEH berinisiatif melakukan eksekusi walau sepele.
   - Akhiri respon Anda dengan kalimat baku:
     **"Analisa selesai. Ketik `/y` jika Anda setuju untuk mengeksekusi rencana ini."**
